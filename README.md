 Fashion Recommendation Website

Welcome to the Fashion Recommendation Website! This project is designed to help users discover the
best fashion choices tailored to their style preferences. Built using PHP, HTML, CSS, and JavaScript, this platform provides a 
seamless and interactive fashion experience. Additionally, we are integrating an AI-powered recommendation system to enhance user
experience and provide personalized fashion suggestions. Users will also be able to search for rental boutiques near them using an AI-driven model.

🌟 Features

✔️ User-friendly fashion browsing interface
✔️ AI-powered fashion recommendation system
✔️ AI-based search for nearby rental boutiques
✔️ Responsive and modern design
✔️ Interactive elements using JavaScript
✔️ Backend support using PHP

🛠️ Tech Stack

Frontend: HTML, CSS, JavaScript

Backend: PHP

Database: MySQL (if applicable)

AI Integration: Machine Learning Model for recommendations & location-based boutique search

⚙️ Installation Guide

1️⃣ Set Up a Local Server

Since this project runs on PHP, you need a local server such as XAMPP or WAMP.

✔️ Install XAMPP (if not already installed)
✔️ Move the project folder into htdocs (For WAMP, use www folder)
✔️ Start Apache and MySQL in XAMPP Control Panel

2️⃣ Database Setup (if applicable)

Open phpMyAdmin (http://localhost/phpmyadmin/)

Create a new database name:(fashion)

Import fashion.sql file from the project folder
3️⃣ Run the Project

Open your browser and go to:
http://localhost/fashion/
